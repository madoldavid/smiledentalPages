Staff guide

How to use reports and keep data safe.

Quick start

Login, choose a tab, filter by name/clinician, export CSV for audits, or download ICS for calendar invites.

Data handling

CSV exports contain personal data. Store in encrypted drives only; delete when finished. Follow UK GDPR.

Status updates

Mark appointments as completed/cancelled via the API (future enhancement). For now, updates are done in the DB.

Support

Contact admin@smilebright.com for access issues. Error logs are on the server; restart with npm run dev.

Retention Exported reports should be retained only as long as needed for audits; purge after use to reduce risk.
